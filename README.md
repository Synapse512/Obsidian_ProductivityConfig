### Obsidian Productivity Configuration
This is a minimal config I made specifically for daily writings, tracking my to-do list, and writing/storing things for school. 

<img width="504" height="336" alt="image" src="https://github.com/user-attachments/assets/5b48c6a8-7b79-4e1d-92bc-ff1b9f2e7c5e" />

---

Features:
* Creates a folder called `_assets` in every vault to store attachments in

Plugins:
* Excalidraw - for writing and drawing notes
* Calendar - tracking daily notes
* File Explorer Note Count - makes it easier to know size of your folders
* Iconize - allows for easy organization of folders by icons
* Ninja Cursor - allows for easier sight of cursor (also feels good)
* Timer - helps with staying focused for certain periods of time (if used for that)
